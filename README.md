## How to run
<ol>
    <li>composer install</li>
    <li>copy env.example content to .env file</li>
    <li>php artisan migrate</li>
    <li>php artisan key:generate</li>
    <li>php artisan storage:link</li>
    <li>npm install</li>
    <li>npm run dev</li>
    <li>php artisan serve</li>
</ol
